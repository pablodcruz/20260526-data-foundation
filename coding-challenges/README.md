# Coding Challenges

Use this folder for iMocha-style Python practice.

## Files

- `coding-challenges.md`: Practice set 1 problem statements.
- `coding-challenges-2.md`: Practice set 2 problem statements.
- `coding_challenge_checks.py`: Starter functions and checkers.

## How to Run

1. Open `coding_challenge_checks.py`.
2. Complete one starter function in the Student Work Area.
3. Print the answer from the function.
4. Run the matching checker (call the function in the checker file and run the file).
